# Enchanted-Movie-Data-Analysis
Data analysis project comparing the movie Enchantes to over 2,000 films using descriptive statistics, a histogram, and a scatter plot. includes comparisons of audience ratings with the minimum, maximum, mean, and median. <br>
Language:Python <br>
Dataset: Rotten Tomatoes Movie Data <br>
Features: <br>
-Compares enchanted audience ratings to each statistic. <br>
-Generates a histogram of audience ratings with each statistic. <br>
-Creates a scatter plot to visualize the data. <br>
Skills: Python, data analysis, descriptive statistics, data visualization.
